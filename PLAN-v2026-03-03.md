# Plan v2026-03-03 — Opportunity Matching Agent

> **Status**: 🟡 Pending Approval  
> **Approach**: Lean — 3 skills, ~2 weeks, API-first

## Core Goal (Only Thing That Matters)

> Autonomous AI agent that daily scans for opportunities, intelligently matches them to Laurence's profile, and manages the application workflow with calendar integration.

---

## 2026 Context & Constraints

- **Anti-scraping escalated**: BOSS直聘, 拉勾网 use ML-based bot detection — browser automation is fragile
- **APIs are reliable**: 猎聘, 智联, 前程无忧 have official APIs with real-name auth
- **LLM-native matching beats keywords**: Holistic reading of JD + company context > regex scoring
- **Self-improving loops are standard**: Agent learns from 👍/👎 feedback
- **Notification fatigue is real**: One quality digest > 10 real-time alerts

---

## Architecture: 3 Skills Only

| Skill | Purpose | Trigger |
|-------|---------|---------|
| `fetch` | Pull from APIs + RSS + web search | Daily cron @ 8AM |
| `score` | LLM-based match analysis with feedback loop | After fetch |
| `act` | Draft emails, calendar, tracker | On high scores or user command |

**Data Flow:**
```
┌─────────┐    ┌─────────┐    ┌─────────┐
│  FETCH  │ →  │  SCORE  │ →  │   ACT   │
│ (daily) │    │(LLM judge)│   │(as needed)│
└─────────┘    └─────────┘    └─────────┘
     ↑                              ↓
     └────── Feedback loop ←───────┘
```

## Match Scoring (LLM-Native)

**Input:**
- Job description (full text)
- Company context (size, stage, product)
- Laurence's profile

**Output:**
```json
{
  "score": 87,
  "confidence": "high",
  "fits": [
    "Your GEO + education background aligns with their AI tutor product",
    "Vibe Coding matches their 'move fast' culture",
    "Photography skill relevant to their content team"
  ],
  "gaps": [
    "Requires 3+ years PM experience (you have 1)",
    "Mandarin business proficiency preferred"
  ],
  "angle": "Emphasize your unique AI+Education hybrid background and rapid prototyping ability"
}
```

**Self-Improvement:** Laurence rates matches (👍/👎). Agent feeds back to tune prompts.

## Notification Strategy

One daily iMessage at 8:30 AM:

```
🎯 Opportunity Digest — Mar 3, 2026

High Matches (80+):
• [Company] [Role] — Score: 87
  Why: Your GEO + education background aligns with their AI tutor product
  Deadline: Mar 10
  
Medium Matches (60-79):
• ... (2-3 items)

Actions:
Reply "apply 1" → I'll draft the email
Reply "save 1" → Add to tracker + calendar
Reply "skip" → Mark rejected, learn from it
```

Batched. Actionable. Conversational.

## Calendar Integration (macOS)

Only 3 event types:
- **Application deadline** (auto-added for 80+ scores)
- **Follow-up reminder** (7 days after "applied")
- **Interview prep** (user-triggered)

Uses AppleScript (`osascript`).
